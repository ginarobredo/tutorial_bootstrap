Instalación de Bootstrap

Para utilizar Bootstrap, es importante conocer los distintos métodos de instalación y configuración. En esta sección explicamos cómo instalar Bootstrap a través de un CDN, npm o descarga manual, junto con los requisitos y configuraciones necesarias.

Selecciona una de las secciones en el menú de navegación para aprender sobre Bootstrap.

Versiones necesarias

La versión recomendada es **Bootstrap 5.3.0** o superior, ya que incluye mejoras en accesibilidad y compatibilidad.

Rutas para tener en cuenta

Si descargas Bootstrap manualmente, debes asegurarte de enlazar correctamente los archivos CSS y JavaScript en la carpeta adecuada dentro de tu proyecto.

Variables de entorno

Bootstrap no requiere la configuración de variables de entorno para su funcionamiento estándar. Sin embargo, si se usa con npm y un entorno de desarrollo como Webpack, puede ser necesario configurar rutas en el archivo de configuración.

¿Necesita base de datos?

No, Bootstrap es un framework de diseño basado en CSS y JavaScript. No requiere una base de datos para su funcionamiento.

Bootstrap puede instalarse de diferentes maneras, dependiendo de las necesidades del proyecto:

1. Uso del CDN (Recomendado)

Este método permite usar Bootstrap sin necesidad de descargar archivos adicionales. Simplemente añade la siguiente línea en la sección `` de tu archivo HTML:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

Para habilitar los componentes interactivos, también es necesario incluir el siguiente script antes de cerrar el ``:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Este es el método que estamos utilizando en este tutorial.

2. Instalación con npm

Si prefieres gestionar dependencias con npm, puedes instalar Bootstrap mediante el siguiente comando en la terminal:

npm install bootstrap

Después de la instalación, puedes importar Bootstrap en tu archivo JavaScript principal:

import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';

Este método es ideal para proyectos que usan frameworks como React, Vue o Angular.

3. Descarga manual

Si deseas tener una copia local de Bootstrap, puedes descargarlo directamente desde su sitio oficial:

Descargar Bootstrap

Después de descargar los archivos, debes incluir en tu proyecto los archivos CSS y JavaScript necesarios:

<link rel="stylesheet" href="ruta/bootstrap.min.css">
<script src="ruta/bootstrap.bundle.min.js"></script>

Este método brinda más control sobre los archivos, pero requiere actualizaciones manuales.
