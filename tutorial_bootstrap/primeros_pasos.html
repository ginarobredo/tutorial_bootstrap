Primeros Pasos con Bootstrap

Aquí aprenderás cómo dar los primeros pasos con Bootstrap, desde la creación de un archivo HTML básico hasta la implementación de estilos y componentes fundamentales del framework.

Selecciona una de las secciones en el menú de navegación para comenzar a aprender sobre Bootstrap.

Con la opcion CDN

Para comenzar a utilizar Bootstrap en tu proyecto, sigue estos pasos básicos.

  1. Crear un archivo HTML básico

    2. Crea un archivo llamado index.html y copia el siguiente código base:

    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primer proyecto con Bootstrap</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container text-center mt-5">
        <h1>¡Hola Mundo con Bootstrap!</h1>
        <p class="text-muted">Este es mi primer diseño usando Bootstrap 5.</p>
        <button class="btn btn-primary">Haz clic aquí</button>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
  2. Explicación del código
    • Incluye el archivo CSS de Bootstrap mediante CDN en el ``.
    • Se usa una clase .container para centrar el contenido.
    • Se aplica la clase text-muted para estilizar el texto.
    • El botón usa la clase btn btn-primary para darle un estilo predeterminado.
  3. Ejecutar el archivo

    4. Guarda el archivo y ábrelo en un navegador. Deberías ver un mensaje de bienvenida con un botón estilizado.

Usando Bootstrap con npm

Si prefieres usar Bootstrap con un administrador de paquetes, sigue estos pasos:

  1. Asegúrate de tener Node.js y npm instalados en tu sistema.
  2. Abre una terminal y ejecuta el siguiente comando para crear un nuevo proyecto:
    3. mkdir mi-proyecto-bootstrap && cd mi-proyecto-bootstrap
npm init -y
  3. Instala Bootstrap con npm:
    4. npm install bootstrap
  4. Crea un archivo HTML y enlaza Bootstrap en tu código JavaScript:
    5. <script type="module">
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
</script>

Este método es ideal si trabajas con frameworks como React, Angular o Vue.

Usando Bootstrap con descarga manual

Si deseas descargar Bootstrap y alojarlo en tu servidor, sigue estos pasos:

  1. Descarga Bootstrap desde su sitio oficial: getbootstrap.com
  2. Extrae los archivos y colócalos en una carpeta dentro de tu proyecto.
  3. Incluye los archivos CSS y JavaScript en tu HTML:
    4. <link rel="stylesheet" href="ruta/bootstrap.min.css">
<script src="ruta/bootstrap.bundle.min.js"></script>

Este método te da más control sobre las versiones y personalización del framework.
