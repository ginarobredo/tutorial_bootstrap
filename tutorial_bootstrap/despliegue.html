Despliegue de un Proyecto con Bootstrap

Cómo publicar tu proyecto en línea.

1. Despliegue en GitHub Pages

GitHub Pages es una opción gratuita para alojar tu proyecto Bootstrap.

  1. Crea un repositorio en GitHub y sube tu código.
  2. Ve a Settings > Pages.
  3. En la sección Source, selecciona la rama main y la carpeta /.
  4. Guarda los cambios y tu sitio estará disponible en https://tu-usuario.github.io/tu-repositorio/.

2. Despliegue en Netlify

Netlify permite desplegar sitios web con solo arrastrar y soltar.

  1. Crea una cuenta en Netlify.
  2. Arrastra tu carpeta del proyecto a la interfaz de Netlify.
  3. Netlify generará un enlace para acceder a tu sitio.

3. Despliegue en Vercel

Vercel es otra opción sencilla para desplegar proyectos Bootstrap.

  1. Crea una cuenta en Vercel.
  2. Conéctalo a tu repositorio de GitHub.
  3. Selecciona el proyecto y haz clic en Deploy.
