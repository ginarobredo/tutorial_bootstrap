Estructura de un Proyecto con Bootstrap

Organización de archivos y sistema de rejilla en Bootstrap.

Estructura de Archivos y Carpetas

Una estructura recomendada para proyectos con Bootstrap es la siguiente:

/mi-proyecto-bootstrap/
│── index.html       # Página principal
│── about.html       # Otra página de ejemplo
│── assets/
│   ├── css/
│   │   ├── styles.css  # Archivo de estilos personalizados
│   ├── js/
│   │   ├── scripts.js  # JavaScript personalizado
│   ├── img/          # Carpeta para imágenes
│── bootstrap/        # Si se usa la descarga manual
│   ├── css/
│   │   ├── bootstrap.min.css
│   ├── js/
│   │   ├── bootstrap.bundle.min.js

Sistema de Rejilla (Grid System)

Bootstrap usa un sistema de rejilla basado en filas y columnas para organizar los elementos en la página.

Ejemplo de una estructura de tres columnas en Bootstrap:

<div class="container">
    <div class="row">
        <div class="col-md-4 bg-primary text-white p-3">Columna 1</div>
        <div class="col-md-4 bg-secondary text-white p-3">Columna 2</div>
        <div class="col-md-4 bg-success text-white p-3">Columna 3</div>
    </div>
</div>

Contenedores en Bootstrap

Bootstrap ofrece varios tipos de contenedores:

  • .container: Contenedor fijo con márgenes automáticos.
  • .container-fluid: Contenedor que ocupa todo el ancho de la pantalla.
  • .container-lg, .container-md: Contenedores responsivos según el tamaño de la pantalla.

Ejemplo de un contenedor fluido:

<div class="container-fluid bg-light p-5">
    <h2>Este es un contenedor fluido</h2>
    <p>Ocupa todo el ancho disponible en la pantalla.</p>
</div>
